2 Note

  • Today, there are many new repositories in github trending. I will try to write down some good or intresting repositories. But maybe there are some errors in expressing ideas in English. If there are any, forgive me.
  • The most popular repository is ffmpeg-libav-tutorial, a tutorial teach you the knowleadge about video, audio, codec and the usage of FFmpeg. It is writen in C. It is not a long tutorial, but with very intesting content. I will find time to read it later.
  • The second popular repository is Relativ. The short description for this is “Build your own VR headset for $100”. The reason to build this is just the authors couldn’t afford an Oculus(a kind of VR headset). So they, three 16-year-old or so teenager and their math teacher, build one, including code for hardware and software.
  • Google’s new repository is named “rejoiner”. It generates a unified GraphQL schema from gRPC microservices and other Protobuf. I feel it is great, even though I don’t quite understand it.
  • intresting repositories:
    • unimatrix : python script to simulate the display from “The Matrix” in terminal.
    • scratches : a very beautiful simple editor written in Node js.
    • genact : a nonsense activity generator.

2 Say

  • at least do something. There are so many brilliant people. Do something so that you can understand what they are doing and take part in it if have chance.
  • There are so many awesome repositories in github. I found a gate to a totally new world. Settle down and learn something from those repositories.
  • I didn’t feel good in daytime. At about 8:00 p.m., I opened github’s trending page and opened my spacemacs to see if there something catch my eyes and can be writen to my blog. The amazing thing happens. I feel astonished about what others are doing and regain the desire to learn more and do more.
  • So tonight will be beautiful.
  • Have good time, briliant guys.
