  • GoRleaser is a release automation tool for Go projects, the goal is to simpilify the build, release and publish steps while providing varaint customization options for all steps. GoReleaser is built for CI tools; you only need to download and execute it in your build script. You can customize your release process by creating a .goreleaser.yml file.

  • GoReleaser is very easy and convenient to use. Just set your repo token and write config file, then every release version will be generate and release to github just by run gorelease in the root of repo dir.

  • Sematic Versioning 2.0.0

      1. MAJOR version when you make incompatible API changes,
      2. MINOR version when you add functionality in a backwards-compatible manner, and
      3. PATCH version when you make backwards-compatible bug fixes.
    • Additional labels for pre-release and build metadata are available as extensions to the MAJOR.MINOR.PATCH format.
