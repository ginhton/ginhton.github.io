  • 被遗忘也是一件遗憾的事啊 - 夏木友人帐
  • go-lang package encoding/json offers two function to encode/decode json
    • json.Marshal(balala) converts specific struct to json string bytes
    • json.Marshal(bytes, &balala) converts json string bytes to specific struct
  • JSON在python中分别由list和dict组成。 json模块提供了四个功能： dumps, dump, loads, load
    • dumps: 将python中的字典转换为字符串
    • loads: 将字符串转换为字典
    • dump: 将数据写入json文件中
    • load: 把文件打开，并把字符串变换为数据类型
