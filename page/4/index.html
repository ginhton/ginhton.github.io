2 Note

  • C-c C-x C-a Archive the current entry using org-archive-default-command
  • C-c C-x C-s or short C-c $ archive the subtree starting at the cursor position to the location given by org-archive-location.
  • org agenda
  • motion
  • n: next line
  • p: previous line
  • view
  • SPC: display the original location of the item in another window.
  • TAP: go to the original location of the item in another window.
  • RET: go to the original location of the item and delete other window.
  • change display
  • o: delete other window
  • d/w: switch to day/ week view.
  • f/b: go forward/backward in time to display the following org-agenda-current-span days.
  • .: go to today.
  • j: prompt for a date and go there.
  • r/g: recreate the agenda buffer, to reflect the changes.
  • s: save all the org buffers in current Emacs session, and also the locations of IDs.
  • remote editing
  • t: change the TODO state of the item, in the agenda and in the file.
  • C-k: delete the current agenda item along with the entire substree belonging to it in the original Org file.
  • C-c C-w: refile the entry at point.
  • C-c C-s: schedule this item, with prefix arg remove the scheduling timestamp.
  • S-right/S-left: Change the timestamp associated with the current line by one day.
  • I: start the clock one the current item
  • O/X: stop/cancel the previously started clock.
  • J: jump to the running clock in another window.

  • javascript copy array

  • :set number/nonumber : used to set/unset line number

  • repeated task +1y/m/d/h : repeat every year/month/day/hour
