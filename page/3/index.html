  • ODF - Open Document Format for Office Application. 通俗的来说，就是想要提供一种开放的office文档的标准。当然，微软提供了更丰富功能的word等。
  • ODT - OpenDocument Text. 开放文档格式
  • python 模仿了 lisp, 使用C编写的
  • 为什么lisp 语言如此先进
    1958 诞生，现代也不过时
    9种思想
    1. 条件结构
    2. 函数也是一种类型
    3. 递归
    4. 变量的动态类型
    5. 垃圾回收机制
    6. 程序由表达式组成
    7. 符号类型
    8. 代码使用符号和常量组成的树形表示法
    9. 无论什么时候，整个语言都是可用的
  • 任何可以用树来表示的数据，同样可以用XML来表示，反之亦然
  • XML 引入属性的理由，就是为了让XML写法不那么冗长。
  • 报告论文，可以在封面直接写明论文的会议、来源、作者和年代
  • HDMI，常被称作高清晰度多媒体接口，是终结以往影音分离传输的全新接口，其最大传输速度可达5Gb/s，除影像数据外，更可同时传输高达8声道的音讯信号。这种非压缩式的数字数据传输，可有效降低数/类转换所造成的信号干扰与衰减。HDMI是首个支持在单线缆上传输，不经过压缩的全数字高清晰度、多声道音频和智能格式与控制命令数据的数字接口。HDMI源于DVI接口技术，它们主要是以美国晶像公司的TMDS信号传输技术为核心，这也就是为何HDMI接口和DVI接口能够通过转接头相互转换的原因。
  • 目前在高清设备中,主要的接口有DVI、HDMI、VGA接口,其中VGA传输的是模拟视频信号,DVI传播的是数字视频信号,HDMI可以同时传输数字视频信号和数字音频信号。
  • VGA模拟信号的传输比较麻烦，首先是将电脑内的数字信号转换为模拟信号，将信号发送到LCD显示器，由显示器再将该模拟信号转换为数字信号，形成画面展示在大家面前。正因为如此，中间的信号丢失严重，虽然可以通过软件的方法修复部分画面，但是随着显示器的分辨率越高画面就会越模糊。
