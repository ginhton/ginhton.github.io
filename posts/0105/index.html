org-roam

  • installation in spacemacs: https://github.com/syl20bnr/spacemacs/blob/develop/layers/+emacs/org/README.org#org-roam-support

  • org-roam-find-file: create or open note

  • org-roam-capture: create note

  • org-roam-insert: make marked text a link to other note. if that note does not exist, create one

  • org-roam-insert-immediate: make marked text a link to other note, but will not open that note after insertation

  • org-roam: show back link

  • from: https://www.zmonster.me/2020/06/27/org-roam-introduction.html
