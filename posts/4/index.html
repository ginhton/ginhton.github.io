commands of spacemacs

  • SPC + ! : run shell commands
  • SPC f y : show and copy curren file path
  • SPC l s : save layout
  • SPC l L : load layout
  • SPC s s : search currence of a word in current buffer
  • SPC s S : similar as the above
  • SPC v : select outer region
  • SPC x a : align code
